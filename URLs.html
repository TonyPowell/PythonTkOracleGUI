URLs Frame

    
     Mouse < Right-click > in any text box invokes
    the text editor popup-menu.
	 
		  Topic Text Box 
     Subject matter of the URL's contents
	 
		  URL Text Box 
     URL of the webpage to be added.
	 
			Decode % URL Button 
     Translates any percent encoded Hebrew text in the URL to
    the Hebrew characters it represents.
	 
			Create Hyperlink Button 
     Takes the Topic and URL fields and formats a hyperlink that
    can be inserted into an Anki flashcard.
	 
			Add URL to DB Button 
     Adds the Topic and its URL to the webpage table in the 
    database.
